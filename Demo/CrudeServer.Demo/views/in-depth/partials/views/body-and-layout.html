@@layout and @@body

This is used to indicate in a layout that there will be a replacement for the body at @@body and which layout to use with @@layout.
Views are processed from the top to bottom. ie:
// index.html
@@layout layout2.html
<h1>HELLO WORLD</h1>

// layout2.html
@@layout layout.html
<div class="side-menu">@@partial side-menu.html</div>
<div class="content">@@body</div>

// layout1.html
<html lang="en" xmlns="http://www.w3.org/1999/xhtml">
    <head>
        <meta charset="utf-8" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <link rel="icon" type="image/x-icon" href="/media/fav-icon.png">
        <title></title>
    </head>
    <body>
        <div class="h-100 container-fluid container-xxl" id="app">
            <main>
                @@body
            </main>
        </div>
    </body>
</html>

Will result
<html lang="en" xmlns="http://www.w3.org/1999/xhtml">

            <main>

                <div class="content">
                    <h1>HELLO WORLD</h1>
                </div>
            </main>
        </div>
    </body>
</html>
