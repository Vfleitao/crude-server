At times, you have some pages that require specific styles (ie: this page has 2 stylesheet and a script for the code highlight).
For this, there is no need for partials of custom helpers. Using standard inline partials works perfectly. 


    // html head element
    <head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link rel="icon" type="image/x-icon" href="/media/fav-icon.png">
    <title>
    ❴❴viewData.title❵❵
    </title>
    @@partial partials/base-styles.html
    ❴❴> page-styles❵❵
    ❴❴#*inline "page-styles"❵❵
    ❴❴/inline❵❵

    </head>

    // Page that needs custom styles / scripts
    @@ layout.html

    ❴❴#*inline "page-styles"❵❵
    <link href="/css/prism.css" rel="stylesheet">
    <link href="/css/in-depth.css" rel="stylesheet">
    <script defer src="/js/prism.js"></script>
    ❴❴/inline❵❵

    // Will result in
    <head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link rel="icon" type="image/x-icon" href="/media/fav-icon.png">
    <title>
    ❴❴viewData.title❵❵
    </title>
    @@partial partials/base-styles.html
    <link href="/css/prism.css" rel="stylesheet">
    <link href="/css/in-depth.css" rel="stylesheet">
    <script defer src="/js/prism.js"></script>
