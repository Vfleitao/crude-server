Getting started

This is a simple guide to get you started with CrudeServer.
Using CrudeServer is simple and easy.

Start by creating a new .NET Console Project and add the CrudeServer NuGet package (once its available) or use a git-submobule.



And setup a server like this:

public class Program
{
    public static async Task Main(string[] args)
    {
        IServerBuilder serverBuilder = new ServerBuilder();
        serverBuilder
            .SetConfiguration(new ServerConfig()
            {
                Hosts = new List<string> { "http://localhost:9000/" }
            })
            .AddCommandRetriever() // Allows Retrieving the registration
            .AddRequestDataRetriever() // Allows Retrieving the request data
            //Add any other middleware here that needs the data (ie: special parsing, etc)
            .AddCommandExecutor(); // Allows Executing the command

            serverBuilder.AddCommand<HomeCommand>("/", HttpMethod.GET);
            
            IServerRunner server = serverBuilder.Buid();
            await server.Run();
    }
}

Once that is done, you now have a server running and listening on your localhost on port 9000.
